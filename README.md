#Later appdating
This project analyzes the disappearance of foreign technical intern trainees in Japan using panel data.

## Data
- Prefecture-level panel data (47 prefectures × 5 years)
- Raw data is not included due to privacy reasons

## Methodology
- Fixed Effects Model (OLS with prefecture & year dummies)
